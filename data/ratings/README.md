# User Ratings Data Storage
#
# This directory contains user rating data
# Files in this directory:
# - ratings.csv - Contains user ratings (user_id, movie_id, rating, timestamp)
# - user_ratings.json - JSON format rating data
#
# Data fields typically include:
# - user_id - Unique user identifier
# - movie_id - Unique movie identifier
# - rating_score - Star rating (1-5 or 1-10 scale)
# - timestamp - When the rating was submitted
#
# This data is used for:
# - Training the recommendation models
# - Filtering recommendations based on ratings
# - User preference analysis
#
